Blackjack Telnet Server

Program Description

This program hosts blackjack over telnet. It is basically just my Blackjack program with sockets added in to allow a user to play from a remote location.

Features

  • Written in C
  • Utilizes telnet connection through sockets
  • Implements TCP packets
  • Functional programming approach
  • Modularized code via subroutines

Sample Run

kevin@LightningJoe:~/public_html/portfolio/blackjack-telnet$ ./bjack 5000
Waiting for data on port TCP 5000
